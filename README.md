遊戲模式選擇

玩家可選擇：
單人模式：只有主角一人進行冒險。
團隊模式：玩家可以招募其他角色組成隊伍。

角色創建:
主角初始屬性由玩家輸入，並分配技能點（總點數不可超過15，每項屬性不得超過10）。
因為遊戲隨機性，隊伍中的其他角色屬性與類型會隨機生成。

冒險過程:
每個世界都有特定的冒險情節和事件，包括小怪戰鬥、BOSS戰鬥及商店互動。
在冒險途中，可以招募新夥伴加入隊伍。
玩家可以選擇是否與敵人戰鬥，逃跑次數過多可能觸發特殊事件。

戰鬥流程
玩家回合：
玩家隊伍中的每位角色依次行動，每位角色可選擇進行普通攻擊、使用技能或元素技能，或選擇跳過回合。

怪物回合：
怪物會選擇隨機目標進行攻擊、可能觸發暴擊，或被角色成功閃避，戰鬥持續進行，直到：玩家隊伍所有角色陣亡（遊戲失敗）、怪物被擊敗（獲得勝利）。

2. 玩家回合選項
(1)普通攻擊
動作描述：

對怪物進行一次基礎攻擊，傷害計算如下：
傷害=角色的基礎攻擊力，如果觸發暴擊，傷害翻倍;若角色的屬性克制怪物屬性，傷害再翻倍。

暴擊機制：
角色暴擊率取決於敏捷（DEX）屬性：
暴擊機率
=dex*5%

屬性克制：
元素屬性之間的克制關係：
水克火、火克木、木克水
若角色屬性克制怪物屬性，傷害翻倍。

(2)使用普通技能

動作描述：
普通技能是角色專屬的能力，每次使用需要消耗一定的MP。
普通技能的效果包括：
單體傷害（如戰士的「Power Strike」造成高傷害）。
增益效果（如巫師的「Arcane Shield」讓角色無敵一回合）。
恢復效果（如治療者的「Healing Light」回復HP）。

技能使用條件：
MP 必須高於技能的消耗值，否則無法使用。

(3)使用元素技能

動作描述：
元素技能需要消耗2倍的MP
元素技能例如：
對怪物施加高傷害並改變其屬性（如巫師的「Water Blast」將怪物屬性改為水）。
為隊友提供支援（如提升隊友屬性，或回復隊友HP）。

元素技能機制：
如果對敵人使用，會先改變敵人的屬性，使其對下一次克制攻擊更脆弱。
如果對隊友使用，若使用者與目標屬性相同，技能效果翻倍。

(4) 跳過回合玩家選擇不進行任何行動，跳過該角色的回合。

防禦:
角色可以選擇防禦行動，在受到攻擊時減少傷害：
減少 20% 傷害，並消耗一定的護甲值。

狀態效果
角色與怪物都可能受到狀態效果影響ex：
攻擊提升：角色下一次攻擊傷害翻倍。
中毒：每回合結束時扣除固定生命值。
無敵：免疫下一次攻擊。

怪物回合(1) 
普通攻擊怪物隨機選擇一名隊伍成員攻擊。
傷害計算公式：
傷害=max⁡(1,怪物的攻擊力)
(2) 暴擊怪物有固定的暴擊率（25%）。
若觸發暴擊，傷害翻倍。
(3) 閃避機制角色可以嘗試閃避怪物的攻擊，機率基於角色的敏捷值（AGL）
：閃避機率
=角色護甲*100%
​
4. 戰鬥結算

勝利時：
玩家獲得經驗值和金幣。
經驗值累積可提升角色等級，增加HP、MP、攻擊力等基礎屬性。
金幣可用於商店購買裝備或提升屬性。

失敗時：
如果玩家的角色「YOU」陣亡，遊戲結束。

商店系統:
玩家可以花費金幣購買：
提升角色屬性點。
購買角色的元素屬性及其對應技能。

BOSS挑戰:
每個世界的最終目標是打敗BOSS。
BOSS戰會根據玩家選擇的模式及逃跑次數觸發不同的事件。

隨機性:
遊戲包含大量隨機生成事件，包括冒險故事線、隊伍成員等。\

隱藏劇情
多次逃跑觸發的「神之干預」
如果玩家在三次戰鬥中選擇逃跑，程式會觸發以下劇情：
一位「作弊級」的角色(c++老師)會被召喚加入隊伍，一般技能擁有強大的攻擊力，可以輕鬆擊敗敵人。
